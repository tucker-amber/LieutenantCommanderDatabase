
Link to primary dataset: https://public.enigma.com/browse/collection/u-s-department-of-agriculture-export-sales/32cae8bb-b8ad-4132-9082-4de17835d44f

For our primary dataset, we chose data from the U.S. Department of Agriculture (USDA). 
Our data pertains to crops exported from the United States and is collected by the USDA. 
This dataset contains six files (each file = 1 year of data. 2000-2005) that provide a detailed description of crops exported from 
the U.S. Some examples of data included in this set are crop type, country code, country name, week exported, region code, and cost.
Irregularities are present in the "commodity code" column (some are listed as strings and some as numbers).

Link to secondary dataset source: https://comtrade.un.org/data/

The secondary dataset was collected from the United Nations, which provides information on commodity trading between nations.
This dataset contains four files (each file = 1 year of data. 2000, 2001, 2004, 2005. Years 2002 and 2003 were not available) that provide information on imports to the United States. Some examples of data included in this set are commodity name, commodity type, total weight, and exporting country. Irregularities include columns with no values and repeated columns. Some values for netweight are ints and some are NULL values.
